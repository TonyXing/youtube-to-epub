YouTube to EPUB Converter

Transform YouTube videos into readable EPUB books with AI-powered summaries

Python FastAPI Azure OpenAI epub.js

The Problem

  • YouTube videos contain valuable information
  • Watching takes time - hard to skim or search
  • No offline reading experience
  • Difficult to reference specific content later

The Solution

Convert any YouTube video to a well-structured EPUB with:

  • AI-generated summaries
  • Organized chapters
  • Full searchable transcript
  • Read anywhere - online or offline

Key Features

Smart Chapters

Uses video chapters or AI segmentation

AI Summaries

Overview + key takeaways per chapter

In-Browser Reader

Read instantly without downloading

Real-time Progress

Live updates during conversion

How It Works

Paste URL → Preview → Convert → Read/Download

Supports youtube.com, youtu.be, and embed links

Architecture

YouTube URL
    │
    ▼
┌─────────────────────────────────────────┐
│  FastAPI Backend                        │
│  ├── youtube-transcript-api (transcript)│
│  ├── pytubefix (metadata & chapters)    │
│  ├── Azure OpenAI (AI summaries)        │
│  └── EbookLib (EPUB generation)         │
└─────────────────────────────────────────┘
    │
    ▼
EPUB File + In-Browser Reader (epub.js)

EPUB Structure

Video Title.epub
├── Cover Page
│   └── Title, Channel, Duration
├── Summary
│   ├── Overview (2-3 paragraphs)
│   ├── Key Takeaways (bullets)
│   └── Video Information
└── Chapters
    ├── Chapter Summary
    ├── Timestamp
    └── Full Transcript

Tech Stack

  • Backend: Python 3.10+, FastAPI, Uvicorn
  • AI: Azure OpenAI (GPT-4o-mini)
  • YouTube: youtube-transcript-api, pytubefix
  • EPUB: EbookLib (generation), epub.js (reader)
  • Frontend: Vanilla HTML/CSS/JS
  • Streaming: Server-Sent Events (SSE)

Demo

localhost:8000

Quick Setup

# Clone & install
git clone https://github.com/TonyXing/youtube-to-epub.git
cd youtube-to-epub
pip install -r requirements.txt

# Configure Azure OpenAI
cp .env.example .env
# Edit .env with your credentials

# Run
python run.py
